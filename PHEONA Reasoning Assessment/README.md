## Description of Code
1) datasets.py: Produce random sampled datasets for downstream tasks.
2) natural_main.py: Run explanation correctness, unfaithful shortcut, and restoration error response generation.
3) simulated_main.py: Run few-shot and hint-biasing experiments.
4) prompts.py: Prompt templates.
5) utils.py: Shared code, such as phenotype mapping.
6) results.py: Code to generate all results and plots in reasoning paper.
